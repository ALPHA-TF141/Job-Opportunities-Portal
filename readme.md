# Job Opportunities Portal

A responsive Job Opportunities Portal built using:

- HTML5
- CSS3
- JavaScript (ES6)

## Features

- Job Listings
- Search Jobs
- Filter by Company
- Filter by Location
- Pagination
- Dark Mode
- Loading Skeletons
- Error Handling
- Responsive Design

## Run Project

Simply open:

index.html

in your browser.
